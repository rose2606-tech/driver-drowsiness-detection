# Driver Drowsiness Detection System

A real-time driver drowsiness detection system using Python, OpenCV, and facial landmark detection to help prevent road accidents.

## 🚀 Features
- Real-time face detection using webcam
- Eye tracking using facial landmarks
- Eye Aspect Ratio (EAR) calculation
- Alert system when driver is drowsy

## 🛠️ Technologies Used
- Python
- OpenCV
- dlib
- NumPy

## 📂 Project Structure

- app.py - Main application  
- detector.py - Drowsiness detection logic  
- templates/ - HTML files  
- static/ - CSS/JS files  

## ⚙️ How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the project:
```bash
python app.py
```

## ⚠️ Important Note

Download the facial landmark model from:  
http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2

Extract it and place it in the project folder before running the project.
